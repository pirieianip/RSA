 To compile this program, go to the included cmake-build-debug folder in cmd:

    1. type "cmake .."
    2. type "make"
    3. put test file in cmake-build-debug
    4. type "rsa435.exe" to initialize
    5. type "sign.exe s yourtestfile" to encrypt your file.
    6. type "sign.exe v yourtestfile.signed yourtestfile" to decrypt your file.
